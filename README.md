# neovim configuration

## requirement for mac os

```bash
brew install ripgrep
```

## how to install all the plugins?

in neovim invoke: `:PackerInstall`

## required lsp servers

install lsp servers with `:Mason` or `:MasonInstall <name>`

- rust\_analyzer
- clangd
- jdtls
- omnisharp
- pyright
- ltex-ls
- ruff
